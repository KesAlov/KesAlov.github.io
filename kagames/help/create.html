Программирование

Движок игры использует язык программирования html, который является одним из самых распространённых языков программирования для документов.

вот основные элементы для этого языка программирования:


<a href="ссылка">текст</a> - создаёт гиперссылку¹
<br> - определяет конец строки
<button onClick="javascript функция">текст</button>-кнопка
<img src="ссылка" alt="текст при ошибке"> - создаёт изображение по ссылке
<script>код javascript</script> - создаёт скрипт¹ с кодом javascript²
<style>код css</style>-делает стиль на языке программирования css¹
<textarea>текст</textarea>-создаёт текст, который можно редактировать

готовые стили:

<h1>текст</h1> - заголовок
<h2>текст</h2> - подзаголовок
<h3>текст</h3> - подподзаголовок
<p>текст</p>-параграф

javascript

document - контролирует страницу

использование:

document.write("код") - пишет в документе то, что вы напишите в кавычки. Может писать и элементы и текст
document.getElementById("айди") - находит элемент в документе через атрибут id

дополнения:

document.getElementById("айди").value - получает текст из textarea
document.getElementById("айди").названиеатрибута = новое содержимое атрибута - изменяет атрибут у элемента
document.getElementById("айди").названиеатрибута - получает содержимое атрибута
конец списка
window.location.href = "ссылка" - перенаправляет игрока по ссылке
var название переменной = "содержимое переменной" - создаёт новую переменную, которую можно использовать вместо кавычек с содержимым в других элементах кода (только в javascript)
function название-функции(переменные-функции"){
код функции - создаёт функцию, спомошью которой можно соединить несколько элементов кода в один
}
if(условие) {функция при условии} - создаёт условие, при котором активируется функция

связанные функции:

else {код при невыполнении} - активирует другой код при невыполнении функции

условия

название любой переменной - проверяет наличие переменной
название любой переменной == "текст" - проверяет, равняется ли переменная определённому тексту
конец списка

CSS

оформление:

название элемента{
атрибуты
}

атрибуты:

background-color:цвет - изменяет цвет фона элемента
background-image: url('ссылка') - изменяет изображение на фоне элемента на изображение по ссылке
color:цвет-изменяет цвет текста в элементе
font:шрифт - изменяет шрифт в тексте элемента
font-size:цифра-изменяет размер текста в элементе

ВАЖНО!

после каждого атрибута ставьте точку с запятой( ; ).

Разрешения

на вход

  • публичный
    • -доступно всем
  • по ссылке
    • -можно зайти по ссылке
  • по паролю
    • -можно зайти, если ввести пароль доступа
  • приватный
    • -никому не доступно
    Модераторы имеют право на изменение разрешения на вход в игру и изменять пароль, если доступ поставлен по паролю. Только они имеют право давать вам разрешение на публикацию пароля от игры
